#### 3항 조건연산자

```js
(조건) ? exp1 : exp2
() 안에 조건이 참이면 exp1 이 실행되며 거짓이라면 exp2 가 실행된다.


IF 문으로 표현한 조건문

if (a > b){
  c = 10;
}else {
  c = 20;
}

위의 식을 조건 연산자로 고쳐 쓰면 아주간단해진다.
(a > b) ? c = 10 : c = 20;
```
