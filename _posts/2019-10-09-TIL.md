* 대표적인 예 Factorial

```js
function factorial(n) {
    // Base Case
    // n이 0이면 재귀를 더 이상 진행하지 않는다.
    if (n === 0) {
        return 1;
    }
    
    // REcursive Case
    return n * factorial(n-1);
}
```



* 대표적인 예 fibonacci 수열

  ```js
  function fib(n) {
      if(n === 0) {
          return 0;
      }
      if(n === 1) {
          return 1;
      }
      // fib(n) === fib(n-1) + fib(n-2);
      return fib(n-1) + fib(n-2);
  }
  // 0 1 1 2 3 5 8 13 ...
  
  ```
