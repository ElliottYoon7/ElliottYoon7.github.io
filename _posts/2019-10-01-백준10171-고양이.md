## 문제

아래 예제와 같이 고양이를 출력하시오.

## 입력

없음.

## 출력

고양이를 출력한다.



## 예제 입력 1 복사

```

```

## 예제 출력 1 복사

```
\    /\
 )  ( ')
(  /  )
 \(__)|
```





### 풀이 

```js
console.log(

    "\\    /\\\n" +
    " )  ( ')\n" +
    "(  /  )\n" +
    " \\(__)|"
)

```



JS에서 백슬러시를 문자열로 넣고 싶을떄는 백슬러시를 두번 넣어야한다.

줄바꿈을 하고 싶을때는 \n 하면된다.
